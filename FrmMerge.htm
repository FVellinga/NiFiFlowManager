The merge form

Here you merge the flows into one merged flow. The process group is extracted from the data flow  and together with some extracts from the common flow put in the merged flow. This window is a DataGridView.

 

The following operations exists:

Delete Use the delete button to delete one row. You can also select via mouse/keyboard multiple rows and use the keyboard delete key.
Up/Down Up/down buttons to define the extract order.
Set as common The selected row is set a the common flow.
Set as merge The selected row is set as the merged flow.
Execute Execute the merging. All done in memory. No physical file created.
Save Save the merged result to the merged flow.

The most right column of the DataGridView contains a button. It can have the colors gray,  green and red and text:

waiting... Added to the merging process, but not extracted yet
extracted The process group from the data flow is extracted.
failed The extraction failed.
COMMON The is the common flow. The button color indicates if extraction was successful.
MERGED This is the merged flow. The button color indicates if extraction was successful.
